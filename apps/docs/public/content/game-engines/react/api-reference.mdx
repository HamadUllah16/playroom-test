## `InsertCoin(InitOptions, callback)`

Initializes Playroom and handles room creation, player joining, and player customization.

```cs
PlayroomKit.InsertCoin(new PlayroomKit.InitOptions()
{
  maxPlayersPerRoom = 2,
  defaultPlayerStates = new() {
    {"score", -500},
  },
}, () => Debug.Log("Game started!"));
```

## `GetState<T>(string key)`

Retrieves the current value of a game state key.

```cs
string winner = PlayroomKit.GetState<string>("winner");
```
