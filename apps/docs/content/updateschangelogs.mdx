## Breaking Changes

### v2.0.0 Breaking Changes

#### Authentication Changes
```javascript
// Old way (v1.x)
Playroom.init('api-key');

// New way (v2.x)
Playroom.init({
  apiKey: 'api-key',
  authMode: 'anonymous' // or 'authenticated'
});
```

#### State Management Changes
```javascript
// Old way (v1.x)
Playroom.setState('key', value, true); // persistent flag

// New way (v2.x)
Playroom.setState('key', value, { persistent: true });
```

## Deprecation Policy

We follow a clear deprecation timeline:

1. **Feature marked deprecated** - Continue working, warnings added
2. **6 months notice** - Feature remains functional
3. **Migration guide released** - Help transition to new approach
4. **Feature removed** - No longer available

## Support for Older Versions

- **Current version** - Full support and updates
- **Previous version** - Security updates only
- **Older versions** - Best effort support
